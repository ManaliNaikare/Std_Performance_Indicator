Student Performance Indicator (ML)

Description

This project analyzes and predicts student performance using various machine learning regression models based on academic and demographic factors.

---

Objective

- Predict student performance scores
- Compare different regression algorithms
- Identify factors affecting student outcomes

---

Models Used

- Linear Regression
- Ridge Regression
- Lasso Regression
- Decision Tree Regression
- Random Forest Regression
- Gradient Boosting
- K-Nearest Neighbors (KNN)
- Support Vector Regression (SVR)

---

Methodology

- Data preprocessing
- Feature selection
- Train-test split
- Model training and evaluation

---

Results

- Best Model: Gradient Boosting
- Final R² Score: 0.98865

---

Tech Stack

- Python
- Matplotlib
- Pandas
- NumPy
